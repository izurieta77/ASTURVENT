Instrucciones de Configuración

Para conectar Google Sheets API:

  1. Ve a console.cloud.google.com
  2. Crea un proyecto nuevo → Activa Google Sheets API
  3. Credenciales → API Key (restringe a Sheets API)
  4. Copia el ID de tu Spreadsheet desde la URL
  5. Edita las variables en SGM_CONFIG al inicio del HTML
  6. Para escritura, necesitas Google Apps Script como web app
💡 Con solo API Key puedes LEER datos. Para ESCRIBIR (guardar despachos) necesitas Apps Script o un backend propio.
Configuración del Sistema
🔐 PINs de Agentes
PIN de 3 dígitos que cada despachador usa para entrar. Solo números.
🔑 Claves de Acceso por Cliente
Clave que cada cliente usa para entrar a su portal. Mínimo 4 caracteres.
Spreadsheets por Cliente
Configura el Spreadsheet ID y nombre de hoja para cada cliente. El ID lo encuentras en la URL de Google Sheets entre /d/ y /edit.
🤖 Analítica IA
Las llamadas a IA ahora corren por el proxy seguro /.netlify/functions/ia usando OpenAI gpt-4o-mini. La clave se configura en Netlify → Site settings → Environment variables como OPENAI_API_KEY (marcada como Secret). Ya no hay que pegar API keys en el navegador.
